# 常用图表绘制

> PyG2Plot 是完全基于 G2Plot 的封装，并且在数据结构上，完全不做任何二次封装，所以配置文档上完全可以参考 [G2Plot](https://g2plot.antv.vision/) 官方文档，从而降低自己维护成本，以及开发者的学习上手成本。

下面会以一些常用的统计图表，来介绍如何使用 PyG2Plot？
